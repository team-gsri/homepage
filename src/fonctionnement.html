Le fonctionnement des missions GSRI

Les types de missions proposées

Le GSRI propose 3 types de missions sur ses serveurs :

  • Les opérations privées, réservées aux membres du GSRI
  • Les opérations publiques, tous les premiers vendredis du mois
  • Les missions impromptues, non planifiées à l'avance

Les mission privées sont réservées aux opérateurs du GSRI. Si vous souhaitez y participer, vous pouvez vous renseigner sur nos modalités de recrutement.

Les opérations publiques

Les opérations publiques sont des opérations organisées officiellement par le GSRI. La participation à ces opérations nécessite une inscription sur notre Discord afin de pouvoir participer. Cette inscription nous permet de connaitre le nombre de joueurs présents à l'opération, pour pouvoir proposer un contenu adapté en terme de menace. Les inscriptions peuvent se faire jusqu'à 2 heures avant l'opération. En cas d'imprévu, il est possible de se désinscrire après la limite d'inscription, mais cela nécessite de nous envoyer un message afin de nous prévenir, car nous serons occupés à préparer la mission. Les règles sont inscrites sur le canal #inscriptions-missions de notre serveur Discord.

Les missions impromptues

Les missions impromptues sout ouvertes à tout joueur qui souhaite participer. Il n'y a pas de procédure d'inscription, il suffit d'installer les mods et de se connecter au serveur de jeu. La présence d'un membre GSRI n'est pas nécessaire, il est parfaitement possible de se connecter à l'improviste sur le serveur et de vous auto-organiser. Nous tenons toutefois à rappeler que nos missions ne sont pas prévues pour des joueurs solitaires. Le respect des règles du serveur reste nécessaire afin d'assurer le bon déroulement des opérations.

Le meilleur moyen d'organiser une mission impromptue est de vous rentre sur notre serveur Discord et de proposer aux joueurs qui le souhaitent de rejoindre le serveur de jeu avec vous. Attention ! Les missions impromptues pourront être interrompues par les membres du GSRI si nous avons besoin du serveur.
