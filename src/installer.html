Comment installer le pack GSRI

Avant de commencer

Pour pouvoir jouer à des missions coop milsim avec le GSRI quelques éléments sont requis :

  • Un PC respectant la configuration recommandée pour Arma 3
  • Un compte Steam en règle VAC
  • Une licence Arma 3 valide
  • Les DLC sont fortement recommandés, en particulier Apex et Contact
  • Un micro-casque de qualité correcte (budget ~ 20€ pour débuter)
  • Une connexion internet décente et stable (ADSL ou mieux)
  • De l’espace disque disponible :
    • 25 Go pour le jeu de base
    • 5 Go pour les mods obligatoires
    • 30 Go pour les mods optionnels

Obtenir de l'aide

Discord est une plateforme de communication conçue pour les communautés de jeux vidéos. C’est la plateforme principale sur laquelle le GSRI se retrouve et organise ses événements. Avant de débuter votre installation, vous pouvez vous y connecter pour obtenir plus d’informations sur le fonctionnement de notre équipe, demander de l’aide si vous êtes bloqué, ou tout simplement tailler la bavette pendant que l’installation se poursuit.

Se connecter au Discord GSRI

Téléchargements

Pour installer le modpack du GSRI, plusieurs logiciels sont requis. Commencez donc par les télécharger et les installer en cliquant sur les liens suivants :

  • Télécharger le runtime Java
  • Télécharger Arma3Sync
  • Télécharger le client Teamspeak 3
  • Télécharger le client Discord (optionnel)
  • Télécharger Windows Management Framework 5.1 (Windows 7 SP1 ou 8.1 seulement)
