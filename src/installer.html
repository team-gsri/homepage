Installer et jouer

Pour pouvoir jouer à nos missions, quelques éléments sont requis :

  • Un ordinateur adapté à Arma 3
  • Un compte Steam en règle VAC
  • Une licence Arma 3 valide
  • Le DLC Apex est recommandé
  • Un micro-casque de qualité correcte
  • Le logiciel Teamspeak 3 est requis
  • Une connexion internet haut débit
  • De l’espace disque disponible :
    • 25 Go pour le jeu de base
    • 5 Go pour les mods obligatoires
    • 30 Go pour les mods optionnels

Téléchargement

La procédure d'installation est très simple grâce à notre launcher personnalisé :

Télécharger le Gsri Sync sur le Microsoft Store
  • Lancez le GsriSync, il vérifie votre installation
  • Si nécesaire, le GsriSync va vous proposer de mettre à jour votre modpack
  • Une fois synchronisé, vous pourrez vous connecter au serveur vocal et au serveur de jeu

Le programme GsriSync est en version beta ; en cas d'erreur n'hésitez pas à nous contacter pour de l'assistance.

Je demande de l'aide sur Discord

Changer de disque

Vous pouvez installer le GsriSync sur un autre disque :

  • Ouvrir le panneau de configuration « Applications »
  • Rechercher l'application « GsriSync »
  • Cliquer sur le bouton « Déplacer »
  • Choisir le disque de votre choix et valider
